Transformamos las variables categóricas -> codificación One-Hot y orden numérico¶

Las variables categricas necesitan ser pasadas a categricas para poder tratarlas. Hay varias formas de hacerlo, pero las ms comunes suelen ser transformarlas a variables ordinales o realizar la codificacin One-Hot

Poner un orden numrico en ocasiones dificulta la prediccin ya que da diferentes pesos a las distintas categoras de una variable, por ello, esta transformacin la vamos a utilizar para aquellas variables que s sigan un orden en sus categora

Por otro lado, para aquellas que no sigan un orden, utilizaremos la codificacin One-Hot. Este mtodo consiste en crear una nueva variable binaria por cada categoria existente en la variable inicial, donde 1 sern las observaciones que pertenezcan a esa categora y 0 las dems.

En muchas tareas, es comn usar k-1 variables binarias en lugar de k, donde k es el nmero total de categoras. Esto se debe a que estamos aadiendo una variable extra redundante que no es ms que una combinacin lineal de las otras y seguramente afectar de manera negativa al rendimiento del modelo. Adems, al eliminar una variable no estamos perdiendo informacin, ya que se entiende que, si el resto de las categoras contienen un 0, la categora correspondiente es la de la variable eliminada.

  • Se ha seleccionado esta opcin tras probar con todas las variables categricas en codificacin One-Hot y todas en orden numrico

def tidy_corr_matrix(corr_mat):
    '''
    Funcin para convertir una matrix de correlacin de pandas en formato tidy
    '''
    corr_mat = corr_mat.stack().reset_index()
    corr_mat.columns = ['variable_1','variable_2','r']
    corr_mat = corr_mat.loc[corr_mat['variable_1'] != corr_mat['variable_2'], :]
    corr_mat['abs_r'] = np.abs(corr_mat['r'])
    corr_mat = corr_mat.sort_values('abs_r', ascending=False)
    
    return(corr_mat)
